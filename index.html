備忘録的なもの

必要なデータをいちいち探すのは面倒くさい…というずぼらな自分のために、とりあえず作ったメモ書きである。

～以下メモ書き部分～

コマンドプロンプト

Base64変換
certutil -encode 入力ファイル 出力ファイル
ダミーデータ作成
fsutil file createnew ドライブ名:ファイルパス 生成サイズ(Byte)
（入力例）\C:\Users\dummy.txt 1024
※0x00で埋め尽くされたバイナリデータが生成される。

VS Code ショートカットキー

自動整形
Shift + Alt + F
カーソル行の下(上)に空行を挿入
Ctrl (+ Shift) + Enter
カーソル行／選択行を移動(複数行対応)
Alt + ↑/↓
カーソル行／選択行をコピー(複数行対応)
Alt + Shift + ↑/↓
コピー(切り取り)
Ctrl + C(X)
※非選択状態で使用すると行コピー(切り取り)になる
単語の部分削除
Ctrl + BS(Del)
カーソル行の削除
Ctrl + Shift + K
